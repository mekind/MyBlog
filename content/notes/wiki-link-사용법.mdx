**Wiki link**는 노트끼리 연결하는 핵심 문법입니다.

## 기본 문법

본문에 다음과 같이 쓰면 됩니다.

```
[[다른 노트의 제목]]
```

또는 슬러그(파일명)로:

```
[[welcome]]
```

## 동작

- **존재하는 노트**로 연결되면 파란색 링크로 표시됩니다 (예: [[환영합니다]])
- **아직 쓰지 않은 노트**는 회색 점선으로 표시됩니다 (예: [[미래의 글]])
- 연결한 노트의 하단에는 자동으로 **백링크 섹션**이 추가됩니다

## 매칭 규칙

1. 슬러그 정확 일치를 먼저 시도
2. 그 다음 제목 정확 일치 (대소문자/공백 정규화)
3. 둘 다 실패하면 미존재 링크로 표시

## 팁

[[환영합니다]]에서 본 것처럼, 한 노트 안에서 여러 개를 자유롭게 연결할 수 있어요.
